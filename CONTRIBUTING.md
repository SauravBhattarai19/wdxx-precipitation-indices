# Contributing to Global Extreme Precipitation Analysis

Thank you for your interest in contributing to this project!

## Code Structure

- `data_processing/`: Core data processing scripts
- `scripts/`: Utility scripts for data combination
- `visualization/`: Visualization and figure generation scripts

## Development Guidelines

1. Follow PEP 8 style guidelines for Python code
2. Add docstrings to all functions and classes
3. Include comments for complex algorithms
4. Test scripts with sample data before committing
5. Update README.md if adding new functionality

## Reporting Issues

Please use GitHub Issues to report bugs or request features. Include:
- Description of the issue
- Steps to reproduce
- Expected vs actual behavior
- System information (OS, Python version)

## Pull Requests

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Test thoroughly
5. Submit a pull request with a clear description
